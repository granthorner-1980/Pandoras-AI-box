What is AI?

The basics — what AI actually means, the key terms, and how to think about it.

Fundamentals Learned

The one-sentence version

AI (Artificial Intelligence) is software that learns patterns from data and uses those patterns to make decisions or generate outputs — rather than following a fixed set of rules written by a programmer.

Key terms

Machine Learning (ML)

The broader field where systems learn from examples. Instead of telling a computer exactly what to do, you show it thousands of examples and it figures out the pattern itself.

Large Language Model (LLM)

A type of ML model trained on massive amounts of text. It learns the statistical patterns of language well enough to generate coherent, contextually relevant text. GPT-4, Claude, Gemini — all LLMs.

Neural Network

The underlying architecture most modern AI is built on. Loosely inspired by the brain — layers of nodes that pass signals to each other, with the connections between them adjusted during training.

Tokens

LLMs don't read words, they read tokens — chunks of text (roughly 3-4 characters on average). "unbelievable" might be 3 tokens. This matters for pricing and context window limits.

How to think about it

The most useful mental model: an LLM is a very sophisticated autocomplete. Given everything it has seen (its training data) and what you've said so far (the prompt), it predicts the most likely next token. Repeat millions of times, and you get a response.

Key insight AI doesn't "know" things the way a human does. It has compressed patterns from its training data. This is why it can be confident and wrong — it's completing a pattern, not recalling a fact.

What it can and can't do

  • Good at: summarising, drafting, translating, classifying, brainstorming, explaining
  • Bad at: precise arithmetic, real-time info, guaranteed accuracy, truly novel reasoning
  • Depends on: how well you prompt it, the quality of the model, the task type
Watch out for Hallucinations — the model generates plausible-sounding but incorrect information. Always verify factual claims, especially for anything high-stakes.
