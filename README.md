# Pandora's AI Box

A personal AI learnings hub.

This repo replaces my PowerPoint of AI notes. Each topic I've learned (or want to learn) gets its own HTML page, all linked from a single home page. The goal is somewhere I can come back to, reflect on what I already know, and remind myself what I still want to explore.

## What's in here

- `index.html` — home page with two grids: **Learned** and **To Learn**
- `pages/` — one HTML file per topic
- `pages/template.html` — starter template for new pages
- `style.css` — shared dark theme
- `CLAUDE.md` — instructions for AI assistants on how the repo is structured

## Viewing it

Open `index.html` in any browser, or use the VS Code Live Preview extension for a side-by-side editor view.
